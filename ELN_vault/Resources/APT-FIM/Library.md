# APT/FIM Resources

Drop PDFs into `Resources/APT-FIM/PDFs`, then use the local writer `Resources` panel to scan, ingest, summarize, and synthesize them with your selected LLM backend.

## PDF Summary Notes

```dataview
TABLE WITHOUT ID
  file.link as "Summary Note",
  provider as "Provider",
  model as "Model",
  file.mtime as "Modified"
FROM "Resources/APT-FIM/Summaries"
SORT file.mtime DESC
```

## Notes

- PDFs live in `Resources/APT-FIM/PDFs`
- Extracted chunk/index data lives in `Resources/APT-FIM/.index`
- Summary notes are regenerated by the writer when you re-run summarization
